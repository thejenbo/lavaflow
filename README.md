# Lavaflow

## A notes app built with React, Redux, Express, and Firebase

This project was bootstrapped with [Create React App](https://github.com/facebookincubator/create-react-app).

### To run this app:
1. Clone the repo
2. run `yarn`
3. `yarn start`
